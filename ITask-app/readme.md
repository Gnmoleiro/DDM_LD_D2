# iTask-app

## Visão Geral do Projeto

O **iTask-app** é uma aplicação de gerenciamento de tarefas (To-Do List) desenvolvida com foco em multiplataforma, utilizando o framework Ionic e o ecossistema Angular.

## Tecnologias Chave

Este projeto é construído com as seguintes tecnologias:

| Tecnologia | Descrição |
| :--- | :--- |
| **Ionic Framework** | Plataforma de desenvolvimento híbrido (Web, iOS, Android). |
| **Angular** | Framework para construção da interface de usuário e lógica da aplicação. |
| **Node.js** | Ambiente de execução JavaScript, essencial para o gerenciamento de pacotes (npm) e ferramentas de build. |

## Pré-requisitos

Para configurar e rodar o projeto localmente, você precisará ter os seguintes softwares instalados em seu ambiente:

1.  **Node.js e npm (ou Yarn)**: Essenciais para o gerenciamento de dependências e execução de scripts.
2.  **Angular CLI**: Interface de linha de comando para criar, desenvolver e manter aplicações Angular.
3.  **Ionic CLI**: Interface de linha de comando para iniciar, construir e testar aplicações Ionic.

## Instalação e Configuração

Siga os passos abaixo para preparar seu ambiente de desenvolvimento e instalar as dependências do projeto.

### 1. Instalação do Node.js

Baixe e instale a versão LTS (Long-Term Support) mais recente do Node.js:

-   **Link para Download:** [https://nodejs.org/en/download](https://nodejs.org/en/download)

### 2. Instalação Global do Angular CLI

Instale o Angular CLI globalmente via npm:

```bash
npm install -g @angular/cli
```

### 3. Instalação Global do Ionic CLI

Instale o Ionic CLI globalmente via npm:

```bash
npm install -g @ionic/cli
```
